# Kratos selfservice UI react-fastapi

A reference self-service UI for [Ory Kratos](https://www.ory.sh/kratos/), built as a **mountable FastAPI app** that serves a **Vite + React SPA** using [`@ory/elements`](https://github.com/ory/elements).

It is a drop-in replacement for [`oryd/kratos-selfservice-ui-node`](https://github.com/ory/kratos-selfservice-ui-node)
